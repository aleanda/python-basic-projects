# Sales Data Analysis

This project analyzes a simple sales dataset using Python.

## Objective
The goal is to extract useful insights from sales data, such as:
- total revenue
- best-selling product
- number of transactions

## Technologies
- Python (basic)
- CSV file handling

## Features
- Reads data from a CSV file
- Calculates total sales
- Identifies the most sold product
- Counts number of sales

## How to run
1. Make sure you have Python installed
2. Run the script:

```bash
python main.py
